# 引言

## 注意力机制的生物学灵感

注意力机制受到人类视觉系统的启发。当人类观察场景时，不会平等处理所有信息，而是将认知资源集中在最相关的部分。这种选择性关注机制使得我们能够在复杂环境中高效处理信息。

```{admonition} 人类视觉注意力的特点
:class: note

1. **选择性**：关注重要信息，忽略无关细节
2. **动态性**：注意力焦点随时间变化
3. **多尺度**：同时关注局部细节和全局上下文
4. **任务驱动**：注意力模式取决于当前任务
```

## CNN中的注意力需求

传统CNN在处理图像时存在一些局限性：

```{admonition} 传统CNN的局限性
:class: warning

1. **平等对待**：所有通道和空间位置被平等处理
2. **信息冗余**：特征图中包含大量冗余信息
3. **计算浪费**：在不重要的区域消耗计算资源
4. **性能瓶颈**：关键特征可能被淹没在噪声中
```

注意力机制通过动态调整特征的重要性来解决这些问题，使网络能够"关注"更相关的信息。
