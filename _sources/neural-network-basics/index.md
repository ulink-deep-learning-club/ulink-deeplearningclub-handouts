# 神经网络基础

**作者**: Anson, 深度学习社 (与 Kimi K2 0905 合作)  

**日期**: 2025-12-05

## 摘要

本文介绍了神经网络训练的基础概念和术语，包括损失函数、优化算法、过拟合与正则化技术等核心内容，并通过详细讲解全连接神经网络（Fully Connected Networks）的基本原理和实现为后续学习打下基础。在此基础上，本文进一步深入探讨卷积神经网络（CNN）的基本原理和架构，包括卷积层、池化层、LeNet架构等核心内容，通过详细的数学推导和PyTorch实现展示CNN在图像识别任务中的优势，并对比CNN与全连接网络的性能差异。此外，本文还探讨了神经网络中的缩放定律，为理解现代深度学习的发展提供了理论基础。
