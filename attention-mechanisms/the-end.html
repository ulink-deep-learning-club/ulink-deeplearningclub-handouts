总结与延伸#

恭喜！你已经学完了 CNN 中注意力机制的完整内容。从 为什么CNN需要注意力机制？ 中 CNN 平等对待所有特征的局限，到 通道注意力：SE-Net 的通道注意力，再到 空间注意力 的空间注意力，最后到 通道+空间注意力：CBAM 的两者结合——你已经掌握了现代 CNN 中最重要的改进技术之一。

注意力与感受野#

卷积神经网络 的 感受野 中我们学习了感受野——每个输出神经元"看到"的输入区域有多大。在标准 CNN 中，感受野通过堆叠卷积层逐步扩大，且对所有输入是固定的（训练完就不变了）。

注意力机制从两个层面改变了这个图景：

空间注意力让感受野"动起来"#

CBAM 的空间注意力可以根据输入内容动态地强调或抑制不同空间位置。这意味着有效感受野不再是固定的——对某些输入，网络可能只关注中心区域；对其他输入，可能关注边缘。这种输入依赖的感受野调整是标准 CNN 做不到的。

自注意力彻底打破感受野的"渐进"假设#

卷积神经网络 的 感受野 告诉我们，CNN 的感受野是通过逐层堆叠小卷积核逐步扩大的——每层只能看到相邻区域。但自注意力 [WGGH18] 完全不同：它在一个操作内连接所有位置，无论距离多远。

方法

感受野特性

计算复杂度

标准 CNN

固定，随深度线性增长

\(O(k^2 \cdot C \cdot H \cdot W)\)

通道注意力（SE）

不变（作用于通道，不影响空间）

\(O(C^2/r)\)

空间注意力（CBAM）

输入依赖，动态调整

\(O(k^2 \cdot H \cdot W)\)

自注意力（Non-local）

全局，一步到位

\(O(C \cdot H^2 \cdot W^2)\)

关键洞察：注意力机制的发展本质上是一个不断放松 CNN 刚性假设的过程——从"所有位置同等重要"（标准 CNN），到"可以动态调整重要性"（CBAM 空间注意力），再到"所有位置可以直接通信"（自注意力）。

延伸：自注意力与多头注意力#

SE-Net 和 CBAM 是 CNN 中专用的注意力形式。你可能会在阅读论文时遇到更通用的自注意力（Self-Attention）和多头注意力（Multi-Head Attention）——它们是 Transformer 架构的核心，在 NLP 和大语言模型中被广泛使用。

简单来说：

  • 自注意力：让序列中的每个位置都能关注到所有其他位置（“全班同学互相看笔记”）

  • 多头注意力：同时从多个角度计算注意力（“多个老师从不同角度批改同一份作业”）

本章提供的代码目录中包含这两种机制的实现，供你延伸阅读：

自注意力模块实现#
 1import torch
 2import torch.nn as nn
 3import torch.nn.functional as F
 4
 5
 6class SelfAttention(nn.Module):
 7    """
 8    自注意力模块 (Non-local 风格)
 9    
10    核心: 计算所有空间位置两两之间的相似度, 用相似度加权聚合特征
11    实现 QKV 形式: Attention(Q,K,V) = softmax(QK^T/√d)V
12
13    输入: (B, C, H, W)
14    输出: (B, C, H, W) — 带残差连接
15    """
16    def __init__(self, in_channels):
17        super(SelfAttention, self).__init__()
18
19        # 1×1 卷积生成 Q, K, V (降维 Q,K 到 C/8 减少计算量)
20        self.query_conv = nn.Conv2d(in_channels, in_channels // 8, 1)
21        self.key_conv = nn.Conv2d(in_channels, in_channels // 8, 1)
22        self.value_conv = nn.Conv2d(in_channels, in_channels, 1)
23
24        # 可学习的缩放因子, 初始为 0 (先学习局部特征, 再逐步引入全局信息)
25        self.gamma = nn.Parameter(torch.zeros(1))
26        self.softmax = nn.Softmax(dim=-1)
27
28    def forward(self, x):
29        batch_size, C, H, W = x.size()
30        N = H * W  # 空间位置数
31
32        # Q: (B, C/8, N) → (B, N, C/8) — 每个位置一个查询向量
33        proj_query = self.query_conv(x).view(batch_size, -1, N).permute(0, 2, 1)
34        # K: (B, C/8, N) — 每个位置一个键向量
35        proj_key = self.key_conv(x).view(batch_size, -1, N)
36        # V: (B, C, N) — 每个位置一个值向量
37        proj_value = self.value_conv(x).view(batch_size, -1, N)
38
39        # 注意力矩阵: (B, N, N)
40        # energy[i,j] = query_i · key_j^T = 位置 i 与位置 j 的相似度
41        energy = torch.bmm(proj_query, proj_key)
42        # softmax 将每行归一化为概率: attention[i,:] 是位置 i 对所有位置的注意力分布
43        attention = self.softmax(energy)
44
45        # 加权聚合: (B, C, N) — 每个位置的输出是 V 的加权和
46        # out[:,:,j] = sum_k V[:,:,k] * attention[j,k]
47        out = torch.bmm(proj_value, attention.permute(0, 2, 1))
48        out = out.view(batch_size, C, H, W)
49
50        # 残差连接: gamma 控制全局信息的注入强度
51        # gamma=0 时退化为恒等映射, 训练中逐步学习全局依赖
52        out = self.gamma * out + x
53
54        return out
55
56if __name__ == "__main__":
57    x = torch.randn(2, 64, 32, 32)
58    sa = SelfAttention(64)
59    y = sa(x)
60    print(f"Input shape: {x.shape}")
61    print(f"Output shape: {y.shape}")
62    print(f"SelfAttention params: {sum(p.numel() for p in sa.parameters())}")
多头注意力模块实现#
 1import torch
 2import torch.nn as nn
 3
 4
 5class MultiHeadAttention(nn.Module):
 6    """
 7    多头注意力模块
 8
 9    将 Q, K, V 投影到 h 个不同的子空间, 独立计算注意力后拼接
10    MultiHead(Q,K,V) = Concat(head_1, ..., head_h)·W_O
11    其中 head_i = Attention(Q·W_Q_i, K·W_K_i, V·W_V_i)
12
13    使用 1×1 Conv2d 实现投影, 适合 CNN 特征图
14    输入: (B, C, H, W)
15    输出: (B, C, H, W)
16    """
17    def __init__(self, in_channels, num_heads=8):
18        super(MultiHeadAttention, self).__init__()
19
20        self.num_heads = num_heads
21        self.head_dim = in_channels // num_heads
22
23        assert self.head_dim * num_heads == in_channels, "in_channels 必须能被 num_heads 整除"
24
25        # QKV 联合投影: (B, C, H, W) → (B, 3C, H, W)
26        self.qkv = nn.Conv2d(in_channels, in_channels * 3, 1)
27        # 输出投影: 拼接后映射回 C 维
28        self.proj = nn.Conv2d(in_channels, in_channels, 1)
29
30    def forward(self, x):
31        batch_size, C, H, W = x.size()
32        N = H * W
33
34        # QKV 投影并分头
35        # (B, 3C, H, W) → (B, 3, h, d, N) 其中 C = h*d, N = H*W
36        qkv = self.qkv(x).reshape(batch_size, 3, self.num_heads, self.head_dim, N)
37        q, k, v = qkv[:, 0], qkv[:, 1], qkv[:, 2]  # 各 (B, h, d, N)
38
39        # 转置为 (B, h, N, d) — 标准注意力需要 N 维在最后第二维
40        q, k, v = q.transpose(-2, -1), k.transpose(-2, -1), v.transpose(-2, -1)
41
42        # 缩放点积注意力: Attention = softmax(QK^T/√d)
43        # q @ k^T: (B, h, N, d) @ (B, h, d, N) → (B, h, N, N)
44        attn = (q @ k.transpose(-2, -1)) * (self.head_dim ** -0.5)
45        attn = attn.softmax(dim=-1)  # (B, h, N, N)
46
47        # 加权聚合: (B, h, N, N) @ (B, h, N, d) → (B, h, N, d)
48        out = (attn @ v).transpose(1, 2).reshape(batch_size, C, H, W)
49        out = self.proj(out)  # 输出投影
50
51        return out
52
53if __name__ == "__main__":
54    x = torch.randn(2, 64, 32, 32)
55    mha = MultiHeadAttention(64, num_heads=8)
56    y = mha(x)
57    print(f"Input shape: {x.shape}")
58    print(f"Output shape: {y.shape}")
59    print(f"MultiHeadAttention params: {sum(p.numel() for p in mha.parameters())}")

下一步学习方向#

掌握了 CNN 的注意力机制后，你可以向以下方向深入：

  1. Transformer 与 ViT：自注意力机制如何彻底替代卷积（U-Net：图像分割的革命 中也有注意力在分割中的应用）

  2. 大语言模型：GPT、BERT 等模型的核心就是多头自注意力

  3. 多模态注意力：CLIP 等模型如何在图像和文本之间做交叉注意力

  4. 高效注意力：稀疏注意力、线性注意力等旨在降低自注意力 \(O(n^2)\) 复杂度的方法

最后的话#

注意力机制的精髓可以用一句话概括：

不是所有信息都同等重要——学会关注该关注的，忽略该忽略的。

这不仅是深度学习的设计原则，也是高效学习的通用智慧。
