深度学习中的度量学习#

作者：Anson，深度学习社
最后更新：2025-03-07

摘要#

度量学习（Metric Learning）是深度学习中的重要范式，其核心思想是通过学习合适的嵌入空间和距离度量，使相似样本靠近、不相似样本远离。这种方法突破了传统分类方法的封闭集限制，支持开放集识别、小样本学习和增量学习，在人脸识别、图像检索、行人重识别等领域取得了巨大成功。

本讲义系统介绍度量学习的理论基础与实践技术：阐述度量学习相对于传统分类方法的优势与核心动机；从监督方式、变换性质、局部性等多个维度梳理度量学习的分类体系；深入讲解经典损失函数（对比损失、三元组损失）与现代损失函数（N-pair、ArcFace、Proxy Loss）的设计原理；最后提供完整的PyTorch实现、采样策略、评估方案和常见问题解决方案。

学习目标#

完成本讲义学习后，你将能够

  1. 理解度量学习的核心思想：解释为什么需要度量学习，掌握度量空间的数学定义

  2. 掌握分类体系：区分监督/非监督、线性/非线性、全局/局部度量学习，理解其适用场景

  3. 熟练运用损失函数：理解不同损失函数的设计原理，能够根据任务特点选择合适的损失

  4. 掌握采样策略：实现并应用semi-hard mining、在线难例挖掘等采样技术

  5. 完成实际项目：使用PyTorch实现完整的度量学习系统，包括数据处理、模型构建、训练评估

前置要求#

必备知识

本讲义假设你已经熟悉以下内容：

  1. 深度学习基础：理解神经网络训练流程、反向传播、优化器（SGD、Adam）

  2. PyTorch编程：能够使用nn.Module定义模型，熟悉Dataset和DataLoader

  3. 损失函数：理解交叉熵损失、均方误差的基本原理

  4. 数学基础：掌握线性代数（向量运算、内积）、概率论基础

  5. 实践经验：至少完成过1-2个图像分类任务的完整训练

如果你还没有这些基础，建议先学习本系列的其他章节。

课程概览#

学习路径总览

本讲义按照从理论到实践的路径组织，建议按顺序学习：

理论基础 → 方法体系 → 核心算法 → 实战应用

每个部分都包含理论讲解、可视化图表和代码示例，帮助你全面掌握度量学习。

第一部分：导论与核心概念#

从"为什么需要度量学习"出发，分析传统分类方法的局限，介绍度量学习的核心思想和数学基础。

本部分亮点

  • 经典方法的困境：封闭世界假设、类别固定、新类别处理困难

  • 度量学习的优势：开放集识别、相似度推理、增量学习

  • 形式化定义：度量空间公理、嵌入映射、距离度量

  • 历史发展：从线性度量学习到深度度量学习的演进

关键收获：理解"我们不学’这是什么’，我们学’这两个有多像’"的核心思想。

第二部分：分类体系与网络架构#

系统梳理度量学习的方法分类和网络架构，建立清晰的知识框架。

本部分亮点

  • 传统度量学习：RCA、LMNN、ITML、KISSME等经典方法

  • 深度度量学习：Siamese网络、Triplet网络、Quadruplet网络

  • 采样策略演进：随机采样、硬负挖掘、Semi-Hard Mining

  • 网络架构选择：ResNet、Inception、Transformer作为主干的选择指南

关键收获：掌握如何根据任务选择合适的架构和采样策略。

第三部分：核心损失函数设计#

深入理解度量学习最核心的技术——损失函数设计。

本部分亮点

  • 对比损失：推拉机制、间隔设计、梯度分析

  • 三元组损失：相对约束、在线采样、Semi-Hard Mining

  • 高级损失函数：N-pair Loss、ArcFace、Proxy Loss、Center Loss

  • 损失函数对比：基于距离vs基于角度、计算复杂度、训练稳定性

关键收获：理解不同损失函数的设计原理，能够根据任务特点选择合适的损失。

第四部分：实践指南与代码实现#

提供完整的实操教程，从零实现度量学习系统。

本部分亮点

  • 数据加载：类别平衡采样、三元组构造

  • 模型定义：ResNet嵌入网络、Triplet网络包装器

  • 训练循环：完整的训练脚本、学习率调度、检查点保存

  • 评估指标：Recall@K、mAP、t-SNE可视化

  • 常见问题：梯度消失、度量坍缩、过拟合的解决方法

  • 应用案例：人脸识别、行人重识别、图像检索的最佳实践

关键收获：能够独立完成度量学习项目的开发与部署。

为什么学习度量学习？#

技术价值#

度量学习的重要性

  1. 开放集识别能力：处理训练未见过的类别，实际应用中的常见需求

  2. 小样本学习潜力：通过度量迁移，少量样本即可实现高效识别

  3. 相似度推理基础：为推荐系统、检索系统提供底层技术支持

  4. 研究热点持续：顶级会议（CVPR、ICCV、NeurIPS）每年有大量相关论文

应用场景#

度量学习已广泛应用于以下领域：

  • 计算机视觉：人脸识别、行人重识别、图像检索、细粒度分类

  • 推荐系统：商品相似度推荐、用户兴趣匹配

  • 自然语言处理：文本相似度、语义检索

  • 医学影像：病灶检索、病例相似性分析

  • 生物识别：指纹、虹膜、步态识别

与其他主题的关系#

度量学习并非孤立的知识点，而是与其他深度学习技术紧密关联：

  • 与迁移学习：度量学习常利用预训练模型迁移通用视觉知识

  • 与对比学习：两者共享"拉近正样本、推远负样本"的思想，可相互借鉴

  • 与注意力机制：注意力可用于增强度量的判别能力

  • 与自监督学习：对比自监督学习为度量学习提供好的特征初始化

Q1：度量学习和分类任务有什么区别？

核心区别：分类要求样本靠近类中心；度量学习要求同类样本聚集，但不必重合。

Q2：应该使用什么损失函数？

初学者路线：

  1. Triplet Loss（首选）

    • 概念直观，易于理解

    • 大多数场景下表现良好

    • 配合Semi-Hard Mining使用

角度间隔，性能优异

Proxy Loss

与代理比较，计算高效

数学符号说明#

本讲义使用以下数学符号约定：

  • \(x\) 或 \(\mathbf{x}\)：输入样本

  • \(y\)：类别标签

  • \(f_\theta(\cdot)\)：嵌入网络，\(\theta\) 为参数

  • \(z = f_\theta(x)\)：嵌入向量（特征）

  • \(d(\cdot, \cdot)\)：距离函数

  • \(\mathcal{L}\)：损失函数

  • \(m\) 或 \(\alpha\)：间隔（margin）

  • \(B\)：批量大小

  • \(N\)：样本总数

  • \(C\)：类别数

  • \(D\)：嵌入维度
