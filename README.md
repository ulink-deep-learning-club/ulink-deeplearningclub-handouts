# Slides for Deep Learning Club

This repo contains the slides that I made for the Ulink Deep Learning Club.

Audience:

- Students with solid math background
- Students with basic python background

GUIDELINES:

- Consistency
- Clarity
- Illustrative
- Readable
- Supported by reasons

    - including math derivation (if applicable)
    - why 'this' instead of 'that'
    - why 'this' is better than 'that'
    - containing implications (if applicable)
